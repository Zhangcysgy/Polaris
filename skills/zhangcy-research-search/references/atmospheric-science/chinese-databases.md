# 中文气象文献搜索指南

## 现状说明

截至目前，CNKI（中国知网）和万方数据**没有公开可用的 REST API**。中文气象文献搜索需通过以下方式：

## 搜索方案

### 方案 A：通过 OpenAlex 搜索英文元数据中的中文论文

OpenAlex 索引了大量中文期刊的英文元数据（标题/摘要翻译）：

```
OpenAlex: search "沙尘暴" OR "dust storm" → 过滤 source.language:zh
```

但这只能搜到有英文元数据的论文。

### 方案 B：引导用户手动搜索

当用户需要中文文献时，提供以下搜索指引：

| 数据库 | URL | 说明 |
|--------|-----|------|
| 中国知网 CNKI | https://www.cnki.net | 最全的中文学术数据库 |
| 万方数据 | https://www.wanfangdata.com.cn | 学协会论文、标准 |
| 维普 | http://www.cqvip.com | 中文科技期刊 |
| 中国科学院科技论文预印本平台 | https://chinaxiv.org | 中文预印本 |

## 搜索建议

当用户明确需要中文文献时，AI 应：
1. 告知 CNKI/万方无公开 API
2. 提供搜索关键词建议（中英文对照）
3. 引导用户到 CNKI 网站手动搜索
