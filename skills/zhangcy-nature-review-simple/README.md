# nature-review skill

投稿前自我审稿技能。帮助作者系统性地评估稿件就绪度，区分关键短板与可优化项，合理判断修改到什么程度可以投出。

## 来源

本技能核心框架来自微信公众号"学术趣芽成长指南"文章《如何判断你的论文"可以投了"？不是你改了多少遍，而是要看这几点》(2026-05-12)，整合了结构化审稿工作流和投稿就绪度评估方法。

## 核心原则

- 写作的目标不是完美，而是清晰
- 改了多少遍 ≠ 是否可以投
- 论文不需要完美，但需要"站得住"
- 投稿本身就是写作过程的一部分

## 工作流

1. **问题清晰度检查** — 核心问题是否说清楚了？
2. **结构对齐检查** — 各部分是否都围绕核心问题？
3. **短板扫描** — 有没有尚未处理的关键短板？
4. **加减判断** — 最近的修改是在加还是在减？
5. **陌生感测试** — 能否像读别人的文章一样读它？
6. **投稿心态检查** — 你在犹豫什么？
7. **综合评估** — 投还是不投？
