# 投稿前自检清单

一份可逐项打勾的自检清单，用于快速评估稿件就绪度。

## 一、问题清晰度

- [ ] 能用一两句话说清论文研究的具体问题（不是泛泛的方向）
- [ ] 引言末尾有明确的研究目标陈述
- [ ] 核心问题自始至终保持稳定，没有中途换题
- [ ] 标题准确反映研究问题（不是太宽泛，也不是太狭窄）

## 二、结构对齐

- [ ] 引言提出的问题在讨论中有回应
- [ ] 文献综述服务于核心问题（不是堆砌文献）
- [ ] 方法设计直接针对要回答的问题
- [ ] 结果数据直接支撑核心结论
- [ ] 讨论回到了最初的提问，回答了"所以呢？"
- [ ] 全文读下来有一种"收拢感"，而非散乱感

## 三、短板检查

- [ ] 没有缺失关键推理步骤
- [ ] 所有数据解释是自洽的
- [ ] 没有明显的结构断裂或逻辑跳跃
- [ ] 方法选择有合理理由（不是"因为别人也这么做"）
- [ ] 重要的对照/基线已包含或已说明为何缺失
- [ ] 局限性已坦诚讨论
- [ ] 没有自我感觉"这里有点勉强"的部分

## 四、收束程度

- [ ] 最近的修改是在删减而非增加
- [ ] 没有冗余段落（删掉不影响论证）
- [ ] 尽量简化了表达而非复杂化
- [ ] 每段第一句清楚引出该段核心观点

## 五、陌生读者友好度

- [ ] 同领域陌生人能较快抓住核心问题
- [ ] 读完引言后能预判正文会讲什么
- [ ] 逻辑流程自然，不需要反复回看
- [ ] 没有依赖"只有作者才知道的背景知识"
- [ ] 所有缩写首次出现时有定义
- [ ] 图和表独立于正文可理解（标题和说明完整）
